```
QOpenRouter — OpenRouter AI Chat Client
```

A minimal Python tool for calling OpenRouter AI models via the OpenAI-compatible API. Supports a command-line workflow (read prompt from file → call API → write response to file) and a full Tkinter desktop GUI with model selection, conversation history, and system prompt configuration.

---

## Features

- Calls any model available on OpenRouter via the OpenAI-compatible API
- File-based CLI workflow — edit a text file, run the script, read the output
- Desktop GUI (`openRouterChat-ui.py`) with chat history, model dropdown, and system prompt panel
- Supports all free models (`:free` suffix) on OpenRouter — no cost required
- API key loaded securely from `.env` — never hardcoded
- Timestamped console logging with elapsed time

---

## Requirements

- Python 3.8+
- An OpenRouter API key — get one at [openrouter.ai/keys](https://openrouter.ai/keys)

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/AutonomousQ/QOpenRouter.git
cd QOpenRouter
```

### 2. Install Python packages

```bash
pip install -r requirements.txt
```

### 3. Add your API key

Create a `.env` file in the project root:

```
OPENROUTER_API_KEY=sk-or-your-key-here
```

> The `.env` file is gitignored and never committed.

## Usage

### Desktop GUI

```bash
python openRouterChat-ui.py
```

- Select a model from the dropdown (includes free models)
- Optionally set a system prompt in the settings panel
- Type your message and click **Send**
- Conversation history is displayed in the scrollable chat window
- Click **Clear** to reset the conversation

### Command Line

1. Edit `input/input.txt` with your model and prompt:

```
{model=nvidia/nemotron-3-super-120b-a12b:free}
Explain MoE architecture briefly.
```

2. Run the script:

```bash
python execution/openRouterChat.py
```

3. Read the response in `output/output.txt`.

---

## Input Format

The first line sets the model using `{model=<model_id>}`. Everything after is the prompt.

```
{model=nvidia/nemotron-3-super-120b-a12b:free}
Your prompt goes here.
It can span multiple lines.
```

To switch models, just change the first line. No code edits needed.

## Free Models on OpenRouter

OpenRouter offers several free models (`:free` suffix). Examples:

| Model | Notes |
|---|---|
| `nvidia/nemotron-3-super-120b-a12b:free` | Strong reasoning, 120B params |
| `mistralai/mistral-7b-instruct:free` | Fast, general purpose |
| `meta-llama/llama-3.1-8b-instruct:free` | Llama 3.1, compact |
| `z-ai/glm-4.5-air:free` | GLM 4.5, efficient |

Browse all free models at [openrouter.ai/models](https://openrouter.ai/models?order=pricing&supported_parameters=free).
